# Code of Conduct

Dockyard follows a simple contributor standard:

- Be respectful and constructive.
- Assume good intent, but accept feedback when impact differs from intent.
- Keep discussions focused on the project and its users.
- Do not harass, threaten, or abuse others.
- Report security issues privately using `SECURITY.md`.

Maintainers may remove comments, issues, or pull requests that violate these expectations.
