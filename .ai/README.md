# AI Operating Documentation

This directory contains repository-specific operating material for humans and AI agents.

It is tool-agnostic. Files under `.ai/` are not Codex plugins, skills, or automatically recognized workflows.

## Layout

- `onboarding/` - historical discovery reports, migration notes, open questions, and known risks.
- `playbooks/` - reusable development procedures.
- `prompts/` - ready-to-use AI prompts.
- `templates/` - reusable document templates.
- `checklists/` - concise verification checklists.

Start with the root `AGENTS.md`, then select the relevant playbook.
