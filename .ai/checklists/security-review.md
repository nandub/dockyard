# Security Review Checklist

- Review scope confirmed.
- Review-only versus remediation confirmed.
- Trust boundaries identified.
- Credential and token handling inspected.
- OCI authentication and registry boundary inspected.
- TLS/certificate handling ownership identified.
- Filesystem, archive, and package verification inspected.
- Signature, SBOM, and supply-chain controls inspected.
- Temporary file and subprocess behavior inspected.
- Risks include evidence and uncertainty labels.
- No fixes applied unless explicitly requested.
