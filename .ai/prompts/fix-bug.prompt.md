# Prompt: Fix Bug

Read `AGENTS.md`, then use `.ai/playbooks/bugfix.md`.

Reproduce or inspect the current behavior before editing. Identify expected behavior from source, tests, docs, or user-provided evidence. Classify uncertain facts as Unknown.

Make the smallest safe fix, add regression coverage where practical, run focused validation first, then broader validation as needed. Report any validation that could not be run and any remaining uncertainty.
