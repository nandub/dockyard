# Prompt: Prepare Release

Read `AGENTS.md`, then use `.ai/playbooks/release.md`. Create an execution plan using `.ai/templates/execution-plan.md`.

Inspect git status, version references, changelog, release docs, Makefile, and GitHub workflows before editing. Do not rewrite tags or publish release assets unless explicitly authorized.

Keep source, docs, examples, CI, version metadata, and release notes aligned. Run the relevant release validation commands and report commands, results, skipped checks, unverified items, and remaining risks.
