# Prompt: Implement Feature

Read `AGENTS.md`, then use `.ai/playbooks/feature.md`. If the work changes CLI contracts, persistent state, OCI behavior, configuration, security posture, or multiple components, create an execution plan using `.ai/templates/execution-plan.md`.

Inspect current behavior and tests before editing. Avoid assumptions and preserve public behavior outside the requested feature.

Implement the smallest scoped change, update tests and docs as needed, run relevant validation, and report summary, files changed, design decisions, tests changed, validation results, unverified items, and remaining risks.
