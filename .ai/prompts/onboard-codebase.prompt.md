# Prompt: Onboard Codebase

Read `AGENTS.md` first, then use `.ai/playbooks/onboarding.md`.

Discover the repository without changing production behavior. Inspect current source, tests, docs, Makefile, and workflows before making claims.

Classify important findings as Verified, Observed, Inferred, or Unknown. Do not present inferred behavior as verified.

Return a concise report with repository structure, architecture boundaries, commands, validation strategy, risks, contradictions, and remaining unknowns. Run only safe read-only commands unless documentation updates are explicitly requested.
